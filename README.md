# Crawler Notebook Updates

This repository contains Jupyter notebooks for crawling and processing data from various Saudi Arabian government agencies and organizations.

## Contents

- **BoG_Library.ipynb**: Notebook for Board of Grievances library crawling
- **GOSI.ipynb**: General Organization for Social Insurance data processing
- **MoI.ipynb**: Ministry of Interior document processing
- **NCAR.ipynb**: National Center for Archives and Records crawling
- **Nezams.ipynb**: Regulations and systems processing
- **Nezams_IDs.09.19.2025.json**: Extracted regulation IDs data

## Data Directories

- **ABBYY_Processed/**: OCR processed documents
- **BOG PDF Crawled 26.09.2025/**: Board of Grievances PDF documents
- **Docx_output/**: Word document outputs
- **GOSI_DOCX/**: GOSI document files
- **moi_pdfs/**: Ministry of Interior PDF files
- **moi_pdfs_ABBYY_Processed/**: OCR processed MoI documents
- **NCAR_PDFs/**: National Center for Archives PDF files
- **Nezams_Docs/**: Regulation documents
- **output/**: Processing output files

## Requirements

- Python 3.x
- Jupyter Notebook
- Required Python packages (install with: pip install -r requirements.txt)

## Usage

1. Clone this repository
2. Install required dependencies
3. Run the relevant Jupyter notebooks for your data processing needs
